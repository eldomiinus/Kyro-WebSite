# Kyro - WebSite
Página web oficial de Kyro, un proyecto comunitario independiente enfocado en ropa y accesorios de estética alternativa.

Este repositorio se encuentra bajo licencia MIT.

En caso de utilizar este código, se agradece la correspondiente atribución a Kyro (@kyroshop.exe).

Kyro no se define como una marca tradicional, sino como una comunidad abierta de personas interesadas en la creación y difusión de diseño independiente. Nuestro objetivo principal es demostrar que los modelos de comercialización actuales dentro de las marcas de ropa son innecesarios al utilizar precios caros para productos baratos, queremos promover una visión más accesible y transparente sobre el valor real de las prendas y accesorios.

El nombre "Kyro" y sus diseños se encuentran disponibles para uso libre. Este repositorio ha sido publicado de manera abierta para fomentar la colaboración, inspirar nuevos proyectos y permitir que cualquier persona pueda participar, replicar o desarrollar ideas similares. Dejando en claro que no tenemos intenciones de restringir o patentar cosas que prácticamente cualquiera puede hacer en casa.

Creemos que la creatividad no debe estar limitada por barreras comerciales innecesarias: cualquiera puede crear y cualquiera puede expresar.

Actualmente, esta página se encuentra en desarrollo. En su estado actual, funciona principalmente como un espacio informativo y de presentación del proyecto, donde se detallan nuestros principios, procesos y objetivos a futuro.

Gracias por su atención. :3
